# Expense Tracker

A simple web app to track your income and expenses with charts, categories, and monthly overview.  
Built with HTML, CSS, and JavaScript. Stores data locally in the browser using LocalStorage.

---

## 🔹 Features

- Add income and expenses with date and category  
- View your **current balance, total income, and total expenses**  
- Manage **custom categories** for expenses  
- **Recent transactions** list  
- **Expense breakdown** pie chart  
- **Monthly overview** bar chart (Income vs Expense)  
- Responsive design for desktop and tablet

---

## 🔹 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (ES6 modules)  
- **Charts:** Chart.js for pie & bar charts  
- **Storage:** LocalStorage (per-user data simulation possible)  
- Optional: Firebase Authentication (can be added later)
